# Безопасность

Никогда не добавляйте в репозиторий:

- токен Telegram-бота;
- `.env`;
- файлы `*.session`;
- `TG_API_HASH`, номер телефона и коды входа;
- локальную папку `.venv`.

Для `BOT_TOKEN` и `BOT_CHAT_ID` используются только GitHub Repository Secrets.

Если токен бота случайно попал в публичный репозиторий, немедленно отзовите
его через `@BotFather` и создайте новый.
