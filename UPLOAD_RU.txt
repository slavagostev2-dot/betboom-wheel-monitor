Исправление ложных уведомлений v2.3

1. В GitHub открой Actions -> Monitor BetBoom wheels.
2. Останови текущий выполняющийся workflow кнопкой Cancel workflow.
3. Распакуй этот архив.
4. Через Add file -> Upload files загрузи ВСЁ содержимое папки одновременно,
   сохраняя папку .github/workflows.
5. Нажми Commit changes.
6. Старые красные workflow не перезапускай.

Первый запуск новой версии:
- только создаст новую базовую точку;
- не отправит старые видимые публикации;
- сохранит state.json;
- затем продолжит проверку примерно каждые 5 минут.

Новое поведение:
- страницы с текстом «Пока ждёшь следующий запуск» считаются неактивными;
- старые Telegram-публикации не рассылаются;
- ночной поиск переносит канал в быстрый список только после подтверждения
  активного колеса;
- ночной workflow больше не запускает лишний второй монитор;
- одинаковые колёса из разных каналов подавляются глобально.
