# Исправление административных кнопок v26

- `Участвую` и `Неактивное` больше не зависят от отдельного промежуточного workflow.
- Изменение `state.json`, `source_health.json` и `source_stats.json` записывается одним атомарным Git commit.
- После сохранения запускается замена непрерывного монитора, чтобы он сразу перечитал новое состояние.
- Бот подтверждает действие только после успешной записи.
- После действия список активных колёс обновляется.
- `Ночное наблюдение` и `Разведка источников` удалены из главного меню и сохранены только внутри раздела `Источники`.
